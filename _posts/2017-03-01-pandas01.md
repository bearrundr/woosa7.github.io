# 데이터 구조

- Series :         1D 동질의 데이터 타입을 갖는 배열(array)
- DataFrame :    2D 테이블 구조. 각 컬럼은 서로 다른 데이터타입을 가질 수 있음.
- Panel :         3D 테이블 구조.


```python
import pandas as pd
import numpy as np
```

## 객체 생성

### 1. Series


```python
s = pd.Series([1,3,5,np.nan,6,8])
s
```




    0    1.0
    1    3.0
    2    5.0
    3    NaN
    4    6.0
    5    8.0
    dtype: float64




```python
s.index
```




    RangeIndex(start=0, stop=6, step=1)




```python
s2 = pd.Series([4, 7, -5, 3], index=['d', 'b', 'a', 'c'])
s2
```




    d    4
    b    7
    a   -5
    c    3
    dtype: int64




```python
s2['a']
```




    -5
